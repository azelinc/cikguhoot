LEARNING OBJECTIVES
-------------------
- Recognise and write numbers up to 100,000 in numerals and words
- Understand place value and digit value
- Partition numbers based on place value and digit value
- Compare and arrange numbers in ascending and descending order

KEY CONCEPTS
------------

1. RECOGNISE AND WRITE NUMBERS

To say a number correctly, say the number in the thousands group first, then 
the next three numbers.

Example: 13,701 is read as "thirteen thousand seven hundred and one"

When writing numbers in numerals, write the digits in the correct order.
When writing numbers in words, use the standard way of reading numbers.

Examples:
- 73,863 = seventy-three thousand eight hundred and sixty-three
- 80,309 = eighty thousand three hundred and nine
- 15,006 = fifteen thousand and six
- 28,050 = twenty-eight thousand and fifty

2. PLACE VALUE AND DIGIT VALUE

Place value is the position of a digit in a number.
Digit value is the value of a digit based on its place value.

In the number 62,307:

  Ten Thousands | Thousands | Hundreds | Tens | Ones
       6        |     2     |    3     |  0   |  7

- Digit 6 is at ten thousands place. Its digit value is 60,000.
- Digit 2 is at thousands place. Its digit value is 2,000.
- Digit 3 is at hundreds place. Its digit value is 300.
- Digit 0 is at tens place. Its digit value is 0.
- Digit 7 is at ones place. Its digit value is 7.

3. PARTITIONING NUMBERS

We can partition numbers in two ways:

a) Based on place value:
   62,307 = 6 ten thousands + 2 thousands + 3 hundreds + 0 tens + 7 ones

b) Based on digit value:
   62,307 = 60,000 + 2,000 + 300 + 0 + 7
   (0 can be left out when partitioning based on digit values)
   62,307 = 60,000 + 2,000 + 300 + 7

4. COMPARING NUMBERS

To compare numbers, follow these steps:
- First, compare the number of digits. The number with more digits is larger.
- If the numbers have the same number of digits, compare the digits starting 
  from the highest place value (ten thousands).

Example: Compare 75,141 and 69,711
- Both are 5-digit numbers.
- Look at the ten thousands digit: 7 and 6.
- 7 has a larger value than 6 (70,000 > 60,000).
- So 75,141 is greater than 69,711.

5. ARRANGING NUMBERS

Ascending order means arranging numbers from the smallest to the largest.
Descending order means arranging numbers from the largest to the smallest.

Example: 1,105 ; 24,712 ; 69,711 ; 75,141
- Ascending order: 1,105 ; 24,712 ; 69,711 ; 75,141
- Descending order: 75,141 ; 69,711 ; 24,712 ; 1,105

WORKED EXAMPLES
---------------

Example 1: Place Value and Digit Value
What is the place value and digit value of 6 in 62,307?

Answer:
Place value of 6 = ten thousands
Digit value of 6 = 60,000

Example 2: Partitioning
Partition 62,307 based on place value and digit value.

Answer:
Based on place value: 62,307 = 6 ten thousands + 2 thousands + 3 hundreds + 0 tens + 7 ones
Based on digit value: 62,307 = 60,000 + 2,000 + 300 + 7

Example 3: Comparing Numbers
Which is greater, 75,141 or 69,711?

Answer:
Compare the ten thousands digits:
- 75,141 has 7 ten thousands (70,000)
- 69,711 has 6 ten thousands (60,000)
- 70,000 > 60,000, so 75,141 > 69,711

Example 4: Arranging Numbers
Arrange these numbers in ascending order:
1,105 ; 24,712 ; 69,711 ; 75,141

Answer:
Ascending order (smallest to largest): 1,105, 24,712, 69,711, 75,141

Mind Teaser:

Then, say and write the numbers.

KEY VOCABULARY
--------------
- Numerals: The symbols used to write numbers (e.g., 62,307)
- Words: Writing numbers using letters (e.g., sixty-two thousand three hundred and seven)
- Place value: The position of a digit in a number (ones, tens, hundreds, thousands, ten thousands)
- Digit value: The actual value of a digit based on its position (e.g., digit 6 in 60,000 has a value of 60,000)
- Partition: Breaking a number into parts based on place value or digit value
- Ascending order: Arranging numbers from smallest to largest
- Descending order: Arranging numbers from largest to smallest
- Compare: Finding which number is greater or smaller
